---

Vencord is an open-source Discord client modification that injects directly into the Discord desktop app or runs in your browser via an extension or UserScript. It adds over 100 quality-of-life plugins, a built-in CSS editor, settings synchronization, and privacy protections — all without any telemetry of its own.

<Warning>
  Client modifications technically violate Discord's [Terms of Service](https://discord.com/terms). Discord has not taken action against users of client mods, and all built-in Vencord plugins are designed to be safe to use. However, if your account is critical to you, proceed at your own discretion. Never use plugins that implement abusive behavior, and avoid posting screenshots that reveal Vencord in servers where it may get you banned.
</Warning>

## How it works

Vencord patches Discord's bundled JavaScript at runtime using a system of webpack patches. Each plugin declares one or more `find`/`replace` patches that target specific module strings inside Discord's bundle. Because patching happens at load time, most changes take effect immediately without restarting Discord.

The settings UI is injected directly into Discord's native settings panel under a **Vencord** section, with sub-pages for Plugins, Themes, Updater, Cloud, and Backup & Restore.
